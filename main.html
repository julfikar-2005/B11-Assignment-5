Question-2: What is the difference between innerHTML, innerText, and textContent ?

Use innerHTML when content with HTML is needed. Use innerText when only visible text is needed. Use textContent when all text is needed, even hidden text.

Question-3: What is event delegation in the DOM?

Event delegation in the DOM Event delegation is a JavaScript technique where an event listener is added to a parent element to handle events from multiple child elements

Question-4: What is event bubbling in the DOM?

Event bubbling in the DOM Event bubbling is a process where an event starts at one of its innermost components and gradually reaches its parent component.
